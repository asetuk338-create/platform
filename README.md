## 我们的想法

当消费者将决策权极大程度让渡给AI（范式三）时，平台垄断了产品展示的“看门人（Gatekeeper）”权力。技术路线升级之外，**议价能力（Bargaining Power）**会不会在消费者、平台与商家之间进行重分配？短期长期有什么区别？比如传统搜索模型中，搜索成本下降会带来长尾效应，但是在AI代理模式下，AI会不会为了降低试错风险而强化头部集聚？

## 初步调研思路
构建数理模型 + 数值模拟

**猜想**AI代理虽然在名义上消除了消费者的搜索摩擦，但由于赋予了平台极端的议价能力与算法导流权，平台将榨取绝大部分的合作剩余。这迫使商家提高基础零售价，最终可能导致总社会福利和消费者剩余的受损。
